= Docker Debian Ruby

Provides a {ruby_link} focused {debian_link} Docker image with current tooling for development purposes. Great for local development and/or continuous integration builds using the most up-to-date tooling.

toc::[]

== Features

* Installs {jemalloc_link}.
* Installs {ruby_link} with {yjit_link}.
* Updates link:https://rubygems.org[RubyGems].

== Requirements

. link:https://www.docker.com[Docker].

== Setup

To set up the project, run:

[source,bash]
----
git clone https://github.com/bkuhlmann/docker-debian-ruby.git
cd docker-debian-ruby
git checkout 0.2.0
----

== Usage

All versions of this image can be found on
link:https://hub.docker.com/r/bkuhlmann/debian-ruby/tags[Docker Hub]. The image can be pulled as follows:

[source,bash]
----
docker pull bkuhlmann/debian-ruby:latest  # <= Pull latest version.
docker pull bkuhlmann/debian-ruby:0.2.0   # <= Pull specific version.
----

You can also build on top of this image via your `Dockerfile`:

[source,dockerfile]
----
FROM bkuhlmann/debian-ruby:latest  # <= Latest version.
FROM bkuhlmann/debian-ruby:0.2.0   # <= Specific version.
----

== Development

To contribute, run:

[source,bash]
----
git clone https://github.com/bkuhlmann/docker-debian-ruby.git
cd docker-debian-ruby
----

To build, run:

[source,bash]
----
bin/build
----

To use the console, run:

[source,bash]
----
bin/console
----

To deploy the _latest_ version, run:

[source,bash]
----
bin/deploy
----

To automatically calculate the next logical version and deploy it, run:

[source,bash]
----
bin/deploy p
----

Versions are calculated and created via the {milestoner_link} gem. Best practice is to deploy both the latest version and next version. Example:

[source,bash]
----
bin/deploy
bin/deploy p
----

The above will ensure new images are created with the following tags:

* `latest`: This is the most recent stable version which may or may not be ahead of the last version released.
* `<major>.<minor>.<patch>`: This the next logical version as calculated and created by {milestoner_link}.
